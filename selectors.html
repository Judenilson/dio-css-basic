Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por IDs

  • Seletor por Tipo: Busca elementos por uma tag HTML
  • Seteror por ID: Busca elementos através do ID, não devem repetir-se
  • Seletor por Classe: Busca elementos através de classes e podem ser repetidos nas páginas
  • Seletor Universal: Seleciona todos os elementos da página
  • Seletor de Atributo: Seleciona através do atributo, podendo ser usado o valor do atributo também.
  • [atributo = valor] - Seleciona o valor exato do atributo
  • [atributo ~= valor] - Seleciona o valor de uma string separada por espaço
  • [atributo |= valor] - Seleciona o valor de uma string separada por hífen
  • [atributo ^= valor] - Seleciona o valor no início da string
  • [atributo $= valor] - Seleciona o valor no final da string
  • [atributo *= valor] - Seleciona o valor em qualquer lugar da string
