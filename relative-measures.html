Unidades de Medidas Relativas

As unidades relativas são calculadas com base em unidades já conhecidas de um outro elemento, o que faz com que o resultado possa variar de acordo com o valor da propriedade do elemento no qual estamos nos baseando.

Porcetagem

A porcentagem é uma unidade de medida que irá utilizar como referência o elemento pai para se basear, podendo alterar os resultados de acordo com o lugar em que está inserido.

Pixel x Porcentagem

widht: 300px;
widht: 50%;

Elemento pai possui 400px de largura e um font-size de 40 px.

widht: 300px;
widht: 50%;

em

A unidade de medida em, quando ultilizada para definir o tamanho de fonte de um elemento, vai utilizarcomo referência o valor da propriedade font-size do elemento pai no qual está inserido.

Caso seja utilizado a unidade em para outras propriedades que não sejam o tamanho da fonte do elemento, o resultado será em relação ao tamanho da fonte do próprio elemento em que etá sendo definida a propriedade.

font size 10px
font size 2em = 20px
font size 2em = 40px

font size 10px, width 50em = 500px

rem

O rem(root em) tem um funcionamento semelhante à unidade de medida em, porém a diferença é que, enquanto o em utiliza o elemento pai como referência, o rem está relacionado ao tamanho da fonte do elemento raiz, chamado root, que no caso, é a tag html.

font size 1rem = 16px
font size 2rem = 32px
font size 3rem = 48px
font size 10px, width 10rem = 160px, height 4rem = 64px

Viewport Units

Viewport é a área visível de uma página web, podendo variar de dispositivo para dispositivo. As unidades de medida viewport, vão ser calculadas em relação ao tamanho da área em que o usuário está vendo sua página.

  • 1vw = 1% do tamanho da largura da área visível da página.
  • 1vh = 1% do tamanho da altura da área visível da página.
widht: 50%;
widht: 50vw;
widht: 50vh;
widht: 20vmin;
widht: 20vmax;

ex e ch

Raramente utilizadas!

A unidade ex irá utilizar o tamanho da altura do caractere x em minúsculo da fonte do elemento.

O ch tem como referência a largura do caractere 0 da fonte do elemento.

Curiosidade: Ao utilizar ch e a fonte for mono-espaçada estaremos definindo quantos caracteres cabem no espaço.
